# 更新日志

所有重要的更新都会记录在此文件中。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.4.0] - 2026-01-16

### 🎉 重大更新
- **项目重命名**：从"摸鱼看盘"更名为"CodeTrader"，更加低调隐蔽
- **全新图标**：采用更中性的代码风格图标，更好地融入开发环境

### ✨ 新增功能
- **连续添加股票**：进入添加模式后可连续添加多只股票，按 ESC 退出
- **连续删除股票**：进入删除模式后可连续删除多只股票，按 ESC 退出
- **智能排序**：股票列表自动按涨跌幅从高到低排序，实时更新

### 🔧 优化改进
- 优化状态栏显示逻辑，初始状态更明显
- 优化股票添加/删除用户体验，减少重复操作
- 完善开发工具链，添加自动构建脚本

### 📚 文档更新
- 更新 README，说明项目来源和改名原因
- 添加开发脚本文档说明
- 完善安装说明，支持 Release 下载安装

## [1.3.0] - 2024-XX-XX

### ✨ 初始功能
- 实时监控股票价格和涨跌幅
- 状态栏显示重点股票信息
- 侧边栏股票列表视图
- 股票添加/删除/清空管理
- 股票搜索功能（支持代码和名称）
- 可配置更新间隔和自定义样式
- 快捷键支持

## 版本说明

- **Major（主版本号）**：不兼容的 API 修改
- **Minor（次版本号）**：向下兼容的功能性新增
- **Patch（修订号）**：向下兼容的问题修正
